# Flash ROM on Redmi Note 7 Pro (violet)

## Steps:

## 1. **Clean Flash**

   - Boot to Recovery
   - Format data
   - Wipe dalvik/cache, cache and data only
   - Flash ROM zip
   - Reboot System


## 2. **Update ( Dirty Flash)**

   - Boot to recovery
   - Wipe dalvik/cache,cache
   - Flash ROM zip
   - Reboot to system

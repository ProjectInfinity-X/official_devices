Device: OnePlus 7 (Guacamoleb)
Recommended Recovery: https://sourceforge.net/projects/infinity-x/files/guacamoleb/15/gapps/infinity.img/download

Flashing Instructions:

Be on OOS12.1

* Clean Flash:
1. Reboot to recovery [Above mentioned infinity.img]

2. Apply update -> adb sideload

3. adb sideload rom.zip

4. Format data

5. Reboot

* Dirty Flash:
1. Reboot to recovery

2. Apply update -> adb sideload

3. adb sideload rom.zip

4. Reboot

* OTA:
1. Simply upgrade via ota updater (either if pushed or via local update import)

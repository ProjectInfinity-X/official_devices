Device: OnePlus Nord (avicii)
AOSP Recovery: https://sourceforge.net/projects/infinity-x/files/avicii/16/infinity_avicii-recovery.img

Clean Flash:
============
1. Boot to recovery [AOSP Recovery Recommended]

2. Apply update -> adb sideload

3. adb sideload <rom.zip>

4. Format data

5. Reboot


Dirty Flash:
============
1. Reboot to recovery

2. Apply update -> adb sideload

3. adb sideload <rom.zip>

4. Reboot


OTA:
====
1. Simply upgrade via ota updater (either if pushed or via local update import)

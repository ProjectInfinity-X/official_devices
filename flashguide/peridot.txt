Device: POCO F6 (peridot)

Flash Guide for 13/10/2025 build 3.3 - and above

Clean Flash is recommended this time

Download all files from: https://sourceforge.net/projects/infinity-x/files/peridot/16/recovery/recovery-3.3/

---------------- Use only HyperOS 2.0.202+ fw. ----------------

1. Reboot to bootloader by "adb reboot bootloader" and type "fastboot -w"
2. fastboot flash boot boot.img
3. fastboot flash init_boot init_boot.img
4. fastboot flash dtbo dtbo.img
5. fastboot flash recovery recovery.img
6. fastboot flash vendor_boot vendor_boot.img
7. Reboot to recovery by "fastboot reboot recovery"
8. Flash ROM in recovery by "Apply Update > adb sideload > adb sideload /path/to/rom.zip"
9. Format data
10. Reboot system

Clean Flash (Installing for first time from hyperos 2 / another rom):

1. Reboot to bootloader by "adb reboot bootloader" and type "fastboot -w"
2. Flash this recovery.img (https://sourceforge.net/projects/infinity-x/files/peridot/15/recovery.img/download)  by "fastboot flash recovery_ab recovery.img"
3. Reboot to recovery by "fastboot reboot recovery"
4. Flash ROM in recovery by "Apply Update > adb sideload > adb sideload /path/to/rom.zip"
5. Format data
6. Reboot system

Dirty Flash / Update (From existing Infinity-X):

1. Reboot into recovery
2. Flash ROM 
3. Reboot system

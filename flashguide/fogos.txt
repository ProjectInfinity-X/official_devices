
# NOTE: if you want to flash custom rom using a custom recovery follow this steps:
# After temporary boot to recovery don't reboot unless it's specified.

## Flashing Guide
• flash copy-partitions.zip if you are flashing custom rom for the first time.
• Flash custom rom
• Format data
• Flash Gapps (if rom is vanilla)
• (Optional) Install recovery as permanent, then reboot to recovery.
• Reboot and enjoy
